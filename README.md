# 📊 A/B Testing Simulation & Statistical Power Analysis

## 📌 О проекте
В данном репозитории представлена симуляция онлайн-экспериментов (А/Б-тестов) с нуля. Проект направлен на валидацию статистических критериев, оценку их корректности на различных распределениях данных и расчет мощности тестов. 

## 🛠 Стек технологий
* **Python** (Pandas, NumPy)
* **Статистика** (SciPy, Statsmodels)
* **Визуализация** (Seaborn, Matplotlib)
* **Методы:** A/A-тестирование, Bootstrapping, симуляции Монте-Карло .

## 🧠 Что было реализовано
1. **A/A тесты (Проверка FPR):** Симуляция тысяч экспериментов на однородных данных для проверки равномерности распределения p-value и удержания ошибки I рода (False Positive Rate) на уровне заданного $\alpha$ (5%).
2. **A/B тесты (Проверка TPR / Мощности):** Симуляция экспериментов с добавленным минимально детектируемым эффектом (MDE) для оценки ошибки II рода и реальной мощности теста.
3. **Работа с распределениями:** Анализ поведения метрик на нормальных и скошенных (лоннг-тэйл) распределениях, характерных для реальных продуктовых данных (например, выручка или конверсия).

## 📊 Главные выводы
* Продемонстрирована корректность работы статистических критериев: на однородных данных ложноположительные срабатывания не превышают 5%.
* Визуализирована зависимость статистической мощности от размера выборки и дисперсии метрики.
* Наглядно показано, почему для метрик с "тяжелыми хвостами" требуются бОльшие объемы трафика для фиксации эффекта.
